# Kimi-k2 Chatbot AI 🤖

A sleek, conversational chatbot interface built using **Streamlit** and powered by the **Groq API** utilizing the `moonshotai/kimi-k2-instruct-0905` model. This application supports real-time response streaming and persistent session history. Generated by **Vaishnavi**.

---

## 🌟 Features

*   **Real-time Streaming:** Watch the AI generate responses token-by-token for a dynamic chat experience.
*   **Session Memory:** Maintains the context of your conversation within the active session so the AI remembers previous exchanges.
*   **Clean UI:** Built with Streamlit's native chat components (`st.chat_message` and `st.chat_input`) for a responsive, clean layout.
*   **Secure API Handling:** Uses `python-dotenv` to keep your Groq API key safe and decoupled from the codebase.

---

## 🛠️ Prerequisites

Before running the project, ensure you have the following installed:
*   Python 3.8 or higher
*   A Groq API Account (and an API key)
